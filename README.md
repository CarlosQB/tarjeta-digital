#Tarjeta de Presentación Digital

Aplicación web desarrollada con HTML, CSS y JavaScript para compartir información de contacto de forma rápida y accesible. El proyecto permite descargar un contacto en formato vCard, generar un código QR para compartir la tarjeta e instalar la aplicación en dispositivos compatibles.

## Características

- Diseño responsivo para dispositivos móviles y escritorio.
- Descarga del contacto en formato `.vcf` (vCard).
- Generación de código QR para compartir la tarjeta.
- Instalación como Progressive Web App (PWA).
- Accesos directos a WhatsApp y correo electrónico.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/CarlosQB/tarjeta-digital.git
```

Entrar al directorio del proyecto:

```bash
cd tarjeta-digital
```

Abrir el archivo `index.html` en cualquier navegador web.
